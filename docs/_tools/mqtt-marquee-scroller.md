A small hardware which can show mqtt messages.

Message input fomat;
```json
{
  "message": "message"
}
```

Face input format;
```json
{
  "panels": [
    { "t":2 },
    { "t":0 },
    { "t":3, "p":70},
    { "t":3, "p":10}
  ]
}
```
